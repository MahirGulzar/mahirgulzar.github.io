Latest Projects

Voxel Terrain

Generation of customized voxel terrain mesh using Tessellation and displacement mapping

A short project in which the idea was to create customized geometery (plane) with desired number of vertices and elavate them using tessellation and displacement mapping. In contrast to this approach, the default Unity terrain was computationally expensive on low-end devices. As it can be easily visualized that displacement map is updated per frame and so does the plane vertz with considerable performance.

Genetic Evolution of a Multi-Joint Virtual Creature

This is a project demo of Advance Algorithmics course Fall 2017. The virtual creature contains multiple joints and the goal is to find the configuration (angles) of joints so that the end point of the creature (Green cube at the top) touches the target (Red Cube in 3D space). We used unity engine for the simulation. Note: Number of joints are fixed.
